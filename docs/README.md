# 快速入门指南

本指南将帮助你快速上手 SkillMap 项目，从安装到使用的完整流程。

## 1. 项目简介

SkillMap 是一个基于 AI 的个性化技能规划与学习助手，它可以：
- 根据你的专业、目标职业和当前水平，生成个性化的技能树
- 提供实时 AI 教练陪练，帮助你学习和复盘
- 追踪你的学习进度，提供个性化的学习建议
- 适配不同设备，提供一致的用户体验

## 2. 系统要求

- **Node.js**：v16.0 或更高版本
- **npm**：v7.0 或更高版本
- **浏览器**：Chrome、Firefox、Safari、Edge 等现代浏览器
- **网络连接**：需要访问 LLM API（如火山引擎、硅基流动等）

## 3. 安装步骤

### 3.1 克隆项目

首先，将项目克隆到你的本地电脑：

```bash
# 使用 git 克隆项目
git clone https://github.com/your-username/skillmap.git

# 进入项目目录
cd skillmap
```

### 3.2 安装依赖

在项目目录中运行以下命令安装依赖：

```bash
npm install
```

这个命令会下载并安装所有需要的依赖包，可能需要几分钟时间。

### 3.3 配置环境变量

1. 复制环境变量模板文件：

```bash
# Windows
copy .env.example .env

# macOS/Linux
cp .env.example .env
```

2. 打开 `.env` 文件，填写你的 LLM API Key：

```env
# 火山引擎 (Ark)
ARK_API_KEY=your_ark_api_key

# 或者 硅基流动 (SiliconFlow)
SILICONFLOW_API_KEY=your_siliconflow_api_key
```

详细的配置说明请参考 [环境配置指南](ENVIRONMENT_GUIDE.md)。

### 3.4 启动开发服务器

运行以下命令启动开发服务器：

```bash
npm run dev
```

启动成功后，会看到类似这样的输出：

```
VITE v6.0.1  ready in 3.28 s

➜  Local:   http://localhost:3000/
➜  Network: use --host to expose
```

打开浏览器，访问 `http://localhost:3000` 即可开始使用。

## 4. 首次使用

### 4.1 生成技能树

1. 在首页点击 "生成技能树" 按钮
2. 填写以下信息：
   - **专业**：你的专业背景
   - **目标职业**：你想从事的职业
   - **当前水平**：你的当前技能水平
   - **每周投入时间**：你每周能投入的学习时间
   - **已掌握技能**：你已经掌握的技能（可选）
   - **补充说明**：其他需要说明的信息（可选）
3. 点击 "生成技能树" 按钮
4. 等待 AI 生成技能树（可能需要几分钟时间）

### 4.2 探索技能树

技能树生成完成后，你可以：

- **查看技能节点**：点击技能节点查看详细信息
- **开始学习**：点击 "开始学习" 按钮开始学习该技能
- **与 AI 对话**：点击 "与 AI 对话" 按钮与 AI 导师交流
- **追踪进度**：系统会自动追踪你的学习进度

### 4.3 使用 AI 教练

1. 点击任何技能节点
2. 点击 "与 AI 对话" 按钮
3. 在对话框中输入你的问题或学习情况
4. AI 教练会根据你的情况提供个性化的学习建议
5. 对话结束后，系统会更新你的学习进度

## 5. 功能介绍

### 5.1 主要功能

- **技能树生成**：AI 为你生成个性化的技能树
- **实时对话**：与 AI 导师实时交流学习问题
- **进度追踪**：自动追踪你的学习进度
- **职业规划**：提供职业发展建议
- **多设备支持**：适配不同设备尺寸

### 5.2 页面导航

- **首页**：项目介绍和技能树生成入口
- **技能树**：查看和管理你的技能树
- **时间线**：查看你的学习历史和进度
- **生成**：生成新的技能树

## 6. 常见问题

### Q: 生成技能树需要多长时间？

A: 生成技能树的时间取决于 LLM 提供商的响应速度，通常需要 1-3 分钟。

### Q: 我可以生成多个技能树吗？

A: 可以，你可以为不同的职业目标生成多个技能树。

### Q: 技能树的技能是固定的吗？

A: 不是，技能树是根据你的输入动态生成的，每个人的技能树都是独特的。

### Q: 我可以修改技能树吗？

A: 目前不支持手动修改技能树，但你可以通过与 AI 对话来调整学习路径。

### Q: 数据存储在哪里？

A: 数据存储在本地的 SQLite 数据库中，位于 `data/skillmap.db` 文件。

## 7. 故障排除

### 7.1 无法启动服务器

如果服务器无法启动，请检查：
- Node.js 是否已正确安装
- 依赖是否已正确安装
- 环境变量是否正确配置

### 7.2 无法生成技能树

如果无法生成技能树，请检查：
- LLM API Key 是否正确
- 网络连接是否正常
- LLM 提供商是否正常运行

### 7.3 AI 对话无响应

如果 AI 对话无响应，请检查：
- LLM API Key 是否有效
- 网络连接是否稳定
- LLM 提供商是否有服务中断
